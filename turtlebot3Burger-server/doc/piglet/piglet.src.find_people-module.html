source code 

Function that is called when we assume a face is found. Robot waits a bit and checks if an mnm is taken. If no MNMs are taken, his id appended to a list of 'heathens', which lists people who don't want MNMs

back_off(direction)

source code 

This method is called when the robot is driving off the edge. The robot drives back a certain distance (direction is decided by which sensor was triggered). The navigation stack is ignored in this process, to prevent the robot from getting stuck.
